# 本地运行说明 v5.0

## 当前版本

- 当前测试版：`v5.0`
- 当前策略：`单用户优先 -> 用户类型分组回退 -> 全局小时模型兜底 -> 近期锚点修正`
- 历史版本目录：`D:\亿云能源科技售电交易数据库\_负荷预测存档\历史版本`

## 输入输出目录

- 预测输入：`D:\亿云能源科技售电交易数据库\1-1负荷预测输入`
- 预测输出：`D:\亿云能源科技售电交易数据库\1-2负荷预测输出`
- 回测/校验输出：`D:\亿云能源科技售电交易数据库\1-3负荷回测效验`

## 当前模型结构

- 低负荷用户：先走低负荷分类器，再走低负荷回归器
- 普通用户：优先命中 `user_{用户编号}_hour_{小时}` 单用户模型
- 单用户模型不足时：回退到 `property_hour_{小时}` / `energy_hour_{小时}` / `hour_{小时}`
- 预测值生成后：再用最近同小时历史锚点做温和修正，不再完全依赖混合模型原始输出

## 日常使用

### 训练 + 预测

- `训练预测_v5_0.bat`

### 回测

- `回测_v5_0.bat`

回测规则：
- 真实值来自：`1-1负荷预测输入\1.分时段历史用电信息`
- 预测值来自：`1-2负荷预测输出\prediction`
- 输出只生成一个 Excel 文件，不再输出多份 CSV

### 校验

- `校验_v5_0.bat`
