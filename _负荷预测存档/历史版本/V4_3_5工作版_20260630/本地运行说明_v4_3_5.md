## 日常使用

### 训练 + 预测

双击：

- `训练预测_v4_3_5.bat`

输入：

- 起始日期 `YYYY-MM-DD`
- 连续天数，直接回车默认 `1`

### 回测

双击：

- `回测_v4_3_5.bat`

说明：

- 回测一定是拿 `1-1负荷预测输入\1.分时段历史用电信息` 里的真实负荷
- 再和 `1-2负荷预测输出\prediction` 里的预测结果按用户、按小时对齐
- 如果真实值还没到那个日期，脚本会直接拦截，不再输出伪回测结果

### 校验

双击：

- `校验_v4_3_5.bat`

说明：

- 校验不是算准确率
- 校验是检查预测文件是否完整、是否有负值、是否走对模型路由

## 回测指标说明

- `MAE`：平均绝对误差
- `RMSE`：均方根误差
- `MAPE`：百分比误差

注意：

- `MAPE` 只统计真实负荷大于等于 `1.0 kW` 的样本
- 这样可以避免夜间接近 0 负荷时把 `MAPE` 人为放大到失真

## 命令行示例

### 训练 + 预测

```bash
python run_forecast_v4_3_5.py --mode train_predict --start-date 2026-06-27 --days 1
```

### 回测

```bash
python run_forecast_v4_3_5.py --mode validate --start-date 2026-06-27 --days 1
```

### 校验

```bash
python run_forecast_v4_3_5.py --mode verify --start-date 2026-06-27 --days 1
```
