# 本地运行说明 v4.0

## 目标

以后你在本地直接运行，不再依赖我帮你跑数据。

## 输入输出目录

- 输入目录：`D:\亿云能源科技售电交易数据库\1-1负荷预测输入`
- 输出目录：`D:\亿云能源科技售电交易数据库\1-2负荷预测输出`

## 主脚本

- `01_train_v4_0.py`
- `02_predict_v4_0.py`
- `03_validate_v4_0.py`
- `run_forecast_v4_0.py`
- `一键运行负荷预测_v4_0.bat`

## 最简单的用法

双击：

- `一键运行负荷预测_v4_0.bat`

然后输入：

1. 运行模式
2. 起始日期
3. 连续天数

## 支持的场景

### 1. 只跑某一天

例如：

- 起始日期：`2026-07-02`
- 连续天数：`1`

### 2. 连续跑多天

例如：

- 起始日期：`2026-07-02`
- 连续天数：`3`

系统会依次运行：

- `2026-07-02`
- `2026-07-03`
- `2026-07-04`

## 运行模式

### `all`

依次执行：

1. 训练
2. 预测
3. 回测

### `train`

只训练模型

### `predict`

只生成预测结果

### `validate`

只做回测

## 命令行用法

如果你以后想直接命令行运行：

```bash
python run_forecast_v4_0.py --mode all --start-date 2026-07-02 --days 1
```

连续多天：

```bash
python run_forecast_v4_0.py --mode predict --start-date 2026-07-02 --days 5
```

## 注意

1. 真实回测依赖真实负荷数据
   - 如果目标日期的真实负荷还没放进输入目录
   - 回测会提示“数据未到”

2. 预测天气已固定规则
   - 最新天气文件优先

3. 测试目录只保留一个主版本
   - 旧版本已经转移到历史版本目录
