# OPERATING_RULES

## 总规则
- 首页是 TALOS 控制台，不是普通欢迎页
- 原始业务目录先保留，TALOS 层负责导航和知识分发

- 新项目入口优先放到 `Projects/`
- 需要长期复用的规则写入 `System/`
