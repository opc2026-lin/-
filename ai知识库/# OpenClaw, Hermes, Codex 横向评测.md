### 第一、Codex：云端编码专家
**原文要点**：代码不在你的电脑上跑，而是克隆到 OpenAI 的隔离沙箱里执行，本地文件系统完全不被碰触，零安全风险。
**核心武器**：
\- **Agent CMD**：告诉 Codex 代码之外的隐性指示（命名规范、业务规则、架构约束）
\- **Go 模式**：围绕一个目标持续推进直到完成，不再问一句答一句
\- **Hook 生命周期自动化**：会话开始自动加载上下文，写完代码自动格式化，结束自动保存状态
\- **杀手功能**：手机 ChatGPT App 直接连电脑上的 Codex，语音就能批准任务、查看进度

**硬伤**：只能用 OpenAI 模型、离线不可用、Pro 订阅 \$200/月

### 第二、OpenClaw：本地全通道助手
**原文要点**：不是编程工具，是一个本地部署的 AI 助手 Gateway。最大优势是 22 个消息通道（微信、QQ、飞书、Telegram、Discord）全部打通。
**核心能力**：
\- Live Canvas 可视化
\- Voice Wake 语音唤醒
\- Talk Mode 对话模式
\- 数据完全在本地，不出设备

**硬伤**：被爆 9×3 实例有安全漏洞（认证绕过、明文存储 API 密钥），2×6 技能含恶意代码，谷歌和 Anthropic 都发了禁令

### 第三、Hermes：自进化通用 Agent
**原文要点**：最特别的能力是自进化——执行任务→评估结果→抽象规律→精炼成技能。用了 10-20 次类似任务后，执行速度提升 2-3 倍。
**核心能力**：
\- 支持 200+ 模型（Kimi、DeepSeek、通义、MiniMax 等）
\- Attribose RL 系统：把 Agent 执行记录导出为训练数据，用来微调自己的模型

**硬伤**：生态规模小、安装复杂、编码能力不如 Codex 专业、自进化初期质量不可控

### 安全性对比
| 框架 | 安全方案 | 合规性 |

| **Codex** | 云沙箱隔离，代码在 OpenAI 隔离环境运行 | HIPAA 合规，医疗金融行业可用 |
