从 “表前资源开发” 转向 “表后的用能组织”

根据您提供的知识库及当前时间（2026年5月），在能源行业中，“表前”与“表后”市场的划分通常以**用户的电能计量表（关口电表）**&#x4E3A;物理边界。结合您提供的多份资料，这两个概念的解释如下：

### 1. 什么是表前市场？

**表前市场**是指从发电厂到用户电表之间（含电网输配环节）的能源市场。其核心特征是**由电网侧或电源侧主导**，强调**集中式**的能源供应和交易。

* **主体**：发电企业、电网公司、大型储能电站。
* **核心活动**：
  * **省间/省内电力现货交易**：如知识库中提到的“山西、广东、山东、甘肃、蒙西、湖北已正式运行”的省间现货及日前、日内、实时市场。
  * **集中式新能源基地**：大型风光基地的并网与调度。
  * **电力辅助服务**：由电网统一组织的调频、调峰。
* **资源特点**：资源是**集中化、规模化、单一流向**（发电→电网→用户）的。
* **盈利逻辑**：主要依靠**发电成本优势**（如大电厂）或**电网通道价值**。

### 2. 什么是表后市场？

**表后市场**是指用户电能计量表**以内**的能源系统及运营服务。其核心特征是**由用户侧或能源服务商主导**，强调**分布式、互动化、效率优化**。

* **主体**：工商业用户（如医院、学校、商场、工厂、公共机构）、节能服务公司（如亿云能源科技）、售电公司、虚拟电厂运营商。
* **核心活动**（结合您知识库中的业务）：
  * **能源费用托管**：替用户管理表后的用电设备，如中央空调、照明、电梯的节能改造（参照“公共机构能源费用托管实施规程”）。
  * **用户侧储能**：如您之前询问的通过峰谷套利、需量管理赚钱的分布式储能。
  * **虚拟电厂（VPP）**：聚合用户表后的空调、充电桩、储能等**可调节负荷**，参与电网需求响应（参照“多维电力增效能”及“虚拟电厂”资料）。
  * **市场化售电**：代理用户参与电力市场交易，直接实现电价下浮。
  * **中水回用、光伏绿电**：建筑内部的分布式能源与水资源循环系统。
* **资源特点**：资源是**分散化、碎片化、双向互动**（用户与电网可双向通信、买卖电）的。
* **盈利逻辑**：主要依靠**精细化管理**（降低用户能耗）和**数字化聚合**（将用户侧碎片化资源打包变现）。

### 3. 为什么说“主战场”在从表前转向表后？

根据知识库资料，这种转向的原因主要有三点：

1. **表前资源开发趋于饱和**：大规模集中式新能源场站的建设受土地、电网消纳能力限制，投资门槛高，竞争激烈。知识库中提到“全面加快电力现货市场建设，2025年底前基本实现电力现货市场全覆盖”，意味着表前交易规则已经基本定型，增量空间减小。
2. **表后空间巨大且价值未充分挖掘**：知识库指出，“建筑能源和市政路灯照明费用托管市场规模10万亿级别以上”。大量的工商业、公共机构（如医院、学校）表后系统老旧、管理粗放，有极大的节能增效和柔性改造潜力。
3. **技术与政策驱动**：
   * **技术**：物联网、大数据、AI使“数字能源集控中心”（如知识库中“多维电力增效能”及“区合同能源管理试点项目”）能够精准实时调度用户侧分散资源。
   * **政策**：《关于鼓励和支持公共机构采用能源费用托管服务的意见》（2022年）以及各地的“虚拟电厂建设”和“需求响应补贴”政策（如您之前提到的广东、山东、江苏方案），都在鼓励资本和服务进入用户侧。

### 总结对比表

| **维度**   | **表前市场 (Front-of-the-Meter)** | **表后市场 (Behind-the-Meter)** |
| -------- | ----------------------------- | --------------------------- |
| **核心边界** | **发电厂 → 用户电表**                | **用户电表 → 用能设备**             |
| **核心资源** | 集中式电厂、大电网、大型储能                | 分布式储能、空调、光伏、充电桩等可调负荷        |
| **核心目标** | 保障电网安全、提高发电效率                 | 降低用户成本、提升用能体验、促进绿色低碳        |
| **主要玩家** | 五大发电集团、电网公司                   | 节能服务公司、售电公司、虚拟电厂运营商         |
| **盈利方式** | 发电上网、电网输配、辅助服务                | 节能改造分成、售电下浮、需求响应补贴          |
| **市场特征** | 成熟、标准化、集中化                    | 蓝海、碎片化、个性化、需综合服务能力          |
