# EMC 合同能源管理模式的优势与风险有哪些？

根据《国务院办公厅转发发展改革委等部门关于加快推行合同能源管理促进节能服务产业发展意见的通知》（国办发〔2010〕25号），合同能源管理是指节能服务公司与用能单位以契约形式约定节能项目的节能目标，节能服务公司为实现节能目标向用能单位提供必要的服务，用能单位以节能效益支付节能服务公司的投入及其合理利润的节能服务机制。

以下是基于知识库整理出的合同能源管理（EMC）模式的主要**优势**与**核心风险**：

### 一、 EMC模式的优势

| **优势维度**     | **具体说明**                                                    | **关键数据/案例出处**                         |
| ------------ | ----------------------------------------------------------- | ------------------------------------- |
| **零投资、零风险**  | 用能单位无需承担项目实施的资金、技术风险。节能服务公司负责全部投资（设计、采购、安装、运维）。             | 知识库中多个案例均提及：医院/学校“零投入、零风险”，湘雅三医院、医学院等 |
| **节能效益显著**   | 由专业公司采用先进技术和管理，节能率高。合同能源管理项目的节能率一般在**10%-40%**，最高可达**50%**。 | 《区人民医院创建节约型公共机构示范单位案例》、宜昌市政策文件        |
| **管理提升与专业化** | 引入专业团队，实现“专家+管家”式服务。用能单位省心省力，将后勤管理压力转移，可专注于核心业务。            | 焦作市第二人民医院案例、西安儿童医院案例：“业主省心”、“保障有力”    |
| **资产更新与移交**  | 合同期内设备由节能公司维护更新；合同结束后，所有节能设备无偿移交给用能单位，后续收益全部归用户。            | 多家医院及学校的EMC合同条款（如医学院、市医院案例）           |
| **风险控制**     | 节能公司承担技术风险、设备运行风险及运营管理责任（如锅炉作为特种设备的运行风险）。                   | 西安儿童医院案例：风险控制，责任转移给运维公司               |

### 二、 EMC模式的核心风险

| **风险类别**      | **具体表现**                                                                                  | **典型案例与知识库依据**                                  |
| ------------- | ----------------------------------------------------------------------------------------- | ----------------------------------------------- |
| **节能效益确认风险**  | 这是最常见、最致命的“坑”。科学的能源基数是合同关键，如果采用“推导式”或“协商式”认定，缺乏分项计量，一旦人员变动，易导致“结算难”，造成项目失败。               | 《整体智慧式合同能源管理经验总结》指出：**没有科学准确的能耗基数，是项目失败的首要原因**。 |
| **合同期与收益不匹配** | 如果节能服务公司对项目投资巨大（如更换中央空调主机），但合同期过短（如5-8年），可能导致投资无法收回。                                      | 多个招标文件规定：合同期一般5-10年，光伏等长回收期项目不超过25年，否则需专家论证。    |
| **甲方配合与边界模糊** | 1. **用能范围扩大**：甲方私自增加大型设备导致能耗飙升，托管方需承担超额费用。&#xA;2\. **物业不执行**：即使系统智能，物业人员仍手动开启设备，导致节能效果打折。 | 张家港市招标文件：规定了基准调整规则。医院案例：需成立“节能工作联合小组”约束物业。      |
| **设备与通讯协议封锁** | 改造空调系统时，主机厂商可能以“未开放通讯协议”为由，拒绝提供接口或收取高昂的“端口费”，导致群控策略无法实现。                                  | 多个空调改造案例中提及此风险：**通讯协议封锁是实施过程中的常见障碍**。           |
| **政策与外部环境变化** | 1. **电价政策调整**：如“单一制”电价改为“两部制”，导致电费上涨。&#xA;2\. **极端天气**：酷暑或严寒导致空调负荷激增，超出合同基准。              | 解决方式：签订补充协议锁定费用基准，或合同中加入“度日数”调整系数。              |
| **甲方违约风险**    | 甲方企业制度改革、更换领导人不履约、或非法经营导致项目中断。                                                            | 《职业技术学院可研报告》中分析了违约风险，贯穿项目各阶段。                   |

### 总结与建议

对于**用能单位（甲方）**：优势在于零投入、零风险，是启动节能改造的最佳选择。但需防范：**选择可靠的节能服务公司**，并要求在合同中明确双方责任边界、节能量确认方法及动态调整机制。

对于**节能服务公司（乙方）**：收益空间大，但需重点防范：**基准核定失真、甲方增量用能甩锅、设备通讯协议封锁**。应在签约前进行详尽的能源审计与设备摸底，并在合同中强化“调整条款”与“免责条款”。
