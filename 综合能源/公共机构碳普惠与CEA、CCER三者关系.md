## 核心观点

碳普惠、CEA、CCER 这三个词，做公共机构节能降碳的必须搞清楚。中国的碳市场分三层：**CEA（强制市场）是根基，CCER（自愿市场）是核心，碳普惠是覆盖小散减碳行为的补充**。三者正在逐步打通，未来公共机构的节能减碳会从"省电费"升级到"省电费 + 赚碳钱"。

***

## 中国碳市场三层架构

### 第一层：CEA（全国碳排放配额）—— 强制市场

| **属性** | **说明**                    |
| ------ | ------------------------- |
| 性质     | 强制市场                      |
| 机制     | 政府给企业核定排放配额，超标需购买，结余可出售   |
| 参与主体   | 火电、钢铁、水泥、电解铝等高排放行业        |
| 履约成本   | 不买就违法，违约成本巨大              |
| 市场规模   | 截至2026年覆盖约50亿吨碳排放，全球最大碳市场 |

**举例**：企业一年配额1万吨，实际排8000吨 → 多出2000吨可卖；实际排12000吨 → 需买2000吨缺口。

### 第二层：CCER（中国核证自愿减排量）—— 自愿市场

| **属性** | **说明**                           |
| ------ | -------------------------------- |
| 性质     | 自愿市场                             |
| 适用项目   | 光伏、风电、林业碳汇、沼气利用等不产生碳排放但帮别人减了碳的项目 |
| 生成方式   | 经国家指定方法学核证后生成 CCER               |
| 用途一    | 拿到 CEA 市场抵消配额缺口（最高抵5%-10%）       |
| 用途二    | 直接卖给有碳中和需求的企业（如互联网公司）            |

### 第三层：碳普惠 —— 面向公众/中小微主体/公共机构的补充机制

| **属性** | **说明**                 |
| ------ | ---------------------- |
| 性质     | 补充机制                   |
| 覆盖行为   | 坐地铁、节约办公用电、垃圾分类等细小减碳行为 |
| 量化方式   | 减碳行为量化成积分              |
| 当前用途   | 积分兑换商品（价值有限）           |
| 未来方向   | 纳入 CCER 体系或与地方碳市场对接    |

***

## 三者关系一句话

> **CEA 是强制市场的根基，CCER 是自愿市场的核心，碳普惠是覆盖小散减碳行为的补充。**

***

## 碳普惠与 CCER 的关系：未来最重要的机会点

### 路径一：碳普惠 → CCER → 全国碳市场

**机制**：碳普惠减排量经过国家统一方法学核证后转为 CCER，进入全国碳市场交易。

**当前难点**：

* 国家层面还没有统一的碳普惠方法学标准
* 各地碳普惠积分规则不一样
* 数据质量参差不齐
* **目前还不能直接转 CCER，但方向已确定**

**政策信号**：2025年9月国管局公开文件提到：

> "探索通过关盘交易、协议转让等方式打通与碳市场的通道，实现减排价值的转化。"

### 路径二：碳普惠 → 地方碳市场（更快落地）

**机制**：在地方碳市场试点地区，碳普惠减排量可作为地方抵消机制，抵消地方配额的一部分。

**已试点地区**：广东、深圳、北京等有地方碳市场的地区。

**特点**：比直接进全国 CCER 市场更快，**先地方后全国，先试点后推广**。

***

## 碳普惠与 CEA 的关系：间接但重要

CEA 市场的履约主体是高排放企业，每年需要履约买配额或买 CCER 来抵消。碳普惠产生的减排量最终如果纳入 CCER 体系，就可以卖给 CEA 市场的履约企业。

**完整链条**：

```
公共机构节能降碳
    ↓
碳普惠量化成减排量
    ↓
核证成为 CCER
    ↓
卖给 CEA 市场的履约企业
    ↓
公共机构获得碳资产收入
```

> 这条链现在还没有完全打通，但每一环的政策都在推进。

***

## 对公共机构能源费用托管的意义

**核心洞察**：这条链全部打通的那天，就是你手里的托管项目多出一个收入来源的时候。

**当前状态**：

* 托管项目 = 省电费
* 未来 = 省电费 + 赚碳钱

***

## 给同行的三个建议

### 建议一：关注当地碳普惠试点

山东、广东、深圳、北京这几个地方走得比较快。如果你在这些地方做托管，现在就要去了解当地的碳普惠平台怎么接入。

### 建议二：把节能改造的减排量单独计量、单独核算

现在不计量，未来碳普惠转 CCER 的时候你没有数据，就拿不到碳资产收入。

### 建议三：关注国家 CCER 方法学的扩容进度

每次方法学扩容都会有新的减排项目类型被纳入。碳普惠如果被纳入，就是政策红利的集中释放期。

## 总结

| **机制**   | **管什么** | **定位**  |
| -------- | ------- | ------- |
| **CEA**  | 强制履约    | 强制市场的根基 |
| **CCER** | 自愿减排    | 自愿市场的核心 |
| **碳普惠**  | 小散的减碳行为 | 补充机制    |

> 三者正在逐步打通，未来公共机构的节能减碳会从**省电费**升级到**省电费 + 赚碳钱**。

***

## 思考与延伸

1. **碳资产收入的"时间差"机会**：政策链条打通需要时间，但提前布局（计量、接入平台）的人将在政策落地时获得先发优势。
2. **数据是碳资产的前提**：没有单独计量的减排数据，未来无法核证为 CCER。现在的节能改造必须建立数据基线。
3. **地方 vs 全国的路径选择**：碳普惠先进地方碳市场（广东、深圳、北京），再进全国 CCER。做地方业务的同行应优先关注本地政策。
4. **方法学扩容的窗口期**：每次 CCER 方法学扩容都是新类型项目纳入的机会，关注国家 CCER 方法学更新动态。
5. **从"省电费"到"赚碳钱"的商业模式升级**：能源托管服务商需要从单纯的节能服务商升级为"节能 + 碳资产管理"的综合服务商。

## 完整文稿

> 碳普惠、CEA、CCER 这三个词，做公共机构节能降碳的，你必须搞清楚。搞不清楚，你跟客户聊节能降碳的时候就露怯了。今天三分钟只讲一件事：这三个机制到底什么关系。
>
> **先建立整体认知，中国的碳市场分三层。**
>
> **第一层叫 CEA**，中文叫全国碳排放配额，这是强制市场。政府给你的企业核定一个排放配额，比如一年可以排一万吨的二氧化碳。如果你实际只排了8000吨，多出来的2000吨配额可以拿到碳交易所去卖。如果你排了12000吨，那超标了，你就要去市场上买缺口。买配额的钱是实打实的履约成本。所以这个市场参与的主体是火电、钢铁、水泥、电解铝这些高排放行业，他们必须履约，不买就违法。截至2026年，CEA 市场已经覆盖了全国约50亿吨的碳排放，是全球最大的碳市场。
>
> **第二层叫 CCER**，全称是中国核证自愿减排量，这是自愿市场。跟 CEA 的强制不同，CCER 是给那些不产生碳排放但帮别人减了碳的项目用的，比如光伏发电、风电、林业碳汇、沼气利用等等。这些项目经过国家指定的方法学核证之后，可以生成 CCER。生成的 CCER 有两种用途：第一种拿去 CEA 市场抵消配额缺口，最高可以抵5%-10%；第二种直接卖给有碳中和需求的企业，比如互联网公司要做碳中和，可以买 CCER 来抵消自己的排放。
>
> **第三层叫碳普惠**，这是面向公众、中小微主体和公共机构的补充机制。它把细小的减碳行为，比如坐地铁、节约办公用电、垃圾分类，量化成积分。积分可以兑换商品，未来探索纳入 CCER 体系或者跟地方的碳市场对接。
>
> **三层的关系用一句话说：CEA 是强制市场的根基，CCER 是自愿市场的核心，碳普惠是覆盖小散减碳行为的补充。**
>
> 再说碳普惠跟 CCER 的关系，这部分是未来最重要的机会点。现在碳普惠的积分主要用途是兑换商品，价值有限。但它未来最值钱的方向是变成 CCER。怎么变呢？
>
> **路径一**：碳普惠减排量经过国家统一方法学核证后可以转为 CCER，进入全国碳市场交易。这个路径的难点在于国家层面还没有统一的碳普惠方法学标准，各地的碳普惠积分规则不一样，数据质量参差不齐，目前还不能直接转 CCER。但方向已经确定了，2025年9月份国管局的公开文件里已经提到了要"探索通过关盘交易、协议转让等方式打通与碳市场的通道，实现减排价值的转化"。这句话就是政策的信号。
>
> **路径二**：在地方碳市场试点地区，碳普惠减排量可以作为地方抵消机制，跟地方的碳市场对接。比如广东、深圳、北京这些有地方碳市场的地区，碳普惠减排量可以用来抵消地方配额的一部分。这个路径比直接进全国 CCER 市场要快。所以碳普惠到 CCER 不是一步到位的，是先地方后全国，先试点后推广。
>
> 碳普惠跟 CEA 的关系是间接的，但很重要。CEA 市场的履约主体是高排放企业，他们每年需要履约买配额或者买 CCER 来抵消。碳普惠产生的减排量最终如果纳入了 CCER 体系，就可以卖给 CEA 市场的履约企业。这就是碳普惠跟 CEA 的间接连接。
>
> **用一条链说清楚**：公共机构节能降碳 → 碳普惠量化成减排量 → 核证成为 CCER → 卖给 CEA 的履约企业 → 公共机构获得碳资产收入。这条链现在还没有完全打通，但每一环的政策都在推进。
>
> 对做公共机构能源费用托管的同行来说，这条链全部打通了那天，就是你手里的托管项目多出一个收入来源的时候。
>
> **最后给同行三个建议**：
>
> 第一，现在就开始关注你们当地有没有碳普惠试点。山东、广东、深圳、北京这几个地方走得比较快。如果你在这几个地方做托管，现在就要去了解当地的碳普惠平台怎么接入。
>
> 第二，做托管项目的时候把节能改造的减排量单独计量、单独核算。现在不计量，未来碳普惠转 CCER 的时候你没有数据，就拿不到碳资产收入。
>
> 第三，关注国家 CCER 方法学的扩容进度。每次方法学扩容都会有新的减排项目类型被纳入。碳普惠如果被纳入，就是政策红利的集中释放期。
>
> **总结一下：CEA 管强制履约，CCER 管自愿减排，碳普惠管小散的减碳行为。三者正在逐步打通，未来公共机构的节能减碳会从省电费升级到省电费加赚碳钱。**
