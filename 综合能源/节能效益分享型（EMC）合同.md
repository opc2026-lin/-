# 节能效益分享型（EMC）合同

#

##

**甲方（用能单位）**：\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

**乙方（节能服务公司/改造实施方）**：\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_

### 第一条 术语与定义

1.1 **合同能源管理（EMC）**：甲乙双方约定项目节能目标，乙方为实现目标提供服务，甲方以节能效益支付乙方投入及合理利润的机制。

1.2 **节能量**：指由基准能耗量减去改造后实际能耗量得出的节约量。**节能量计算必须依据附件三《节能量测量与验证方案》执行。**

### 第二条 项目期限与效益分享

2.1 **项目期限**：项目建设期为\_\_\_\_日历天；效益分享期为\_\_\_\_年，自项目验收合格之日起算。

2.2 **效益分享比例**：

* 第1-3年：甲方享有\_\_\_\_%，乙方享有\_\_\_\_%；
* 第4-5年：甲方享有\_\_\_\_%，乙方享有\_\_\_\_%。
  &#x20;**乙方分享的节能效益总额不足以弥补乙方投资及合理利润的，甲方应予以补足（或约定保底节能量）。**

### 第三条 节能量测量与验证（核心风控条款）

3.1 **基准能耗的确定**：以双方签字确认的改造前连续\_\_\_\_天的能耗数据为基准。**基准能耗必须排除非正常因素（如设备故障停机、疫情封控等）影响。**

3.2 **节能量计算**：节能量 = 基准能耗 - 校准能耗。

3.3 **不可抗力与外部变化调整**：在效益分享期内，若发生以下情况，必须对基准能耗进行校准调整，否则乙方不承担节能量下降责任：

（1）甲方变更建筑用途、增加大型用能设备（≥15KW）、增加用能面积或人员；

（2）甲方现有设备（未改造部分）老化导致能效衰减；

（3）极端天气导致空调负荷异常增加；

（4）甲方物业人员私自修改控制策略或手动强制开启设备。

3.4 **争议解决**：对节能量有争议时，由双方共同委托具备资质的第三方机构进行审计，费用由责任方承担（若争议差额＜5%，由甲方承担；＞5%由乙方承担）。

### 第四条 改造实施与系统保障

4.1 **施工免责**：乙方应文明施工，尽量不影响甲方正常运营。但**因不可预见的技术原因或甲方提供的图纸/资料与现场严重不符，导致的工期延误或局部停水停电，乙方不承担违约责任，工期顺延。**

4.2 **通讯与接口**：涉及现有系统（如BA、空调主机）接口与通讯协议开放的，**甲方有义务协调原供应商无条件配合开放接口。若因原供应商设置技术壁垒或收取高额端口费导致无法实现原定群控功能的，乙方有权调整技术方案，且节能目标相应下调；已发生的额外费用由甲方承担。**

4.3 **验收**：项目完工后，甲方应在收到验收申请后10个工作日内组织验收。**若甲方无正当理由逾期不组织验收，或实际已投入使用，视为验收合格，效益分享期自投入使用之日起算。**

### 第五条 所有权与风险转移

5.1 **所有权转移**：在效益分享期内，项目财产所有权属于乙方；**分享期满且甲方付清所有应付款项后，项目财产所有权无偿转移给甲方**。

5.2 **设备保障**：托管期内，由乙方投资改造的设备维保由乙方负责；**甲方现有设备的维保及大修由甲方负责。若因甲方现有设备故障导致乙方节能设备无法运行或节能量受损，甲方仍需按约定支付乙方应得的节能效益分享款。**

### 第六条 付款与违约责任

6.1 **付款义务**：甲方应在收到乙方出具的节能效益发票后10个工作日内付款。

6.2 **甲方违约**：

（1）逾期付款的，每日按应付未付金额的万分之五支付违约金；

（2）**甲方未经乙方同意，擅自拆除、停用、篡改节能设备，或变更项目边界导致节能量无法计算的，视为乙方已实现全部预期节能效益，甲方应一次性向乙方支付剩余分享期内的全部预期收益。**

6.3 **合同提前终止**：非因乙方违约导致合同提前解除的，甲方必须按乙方未收回的投资额加上预期利润（不低于年化8%）一次性买断项目资产。
