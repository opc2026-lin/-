# 储能商业模式

好的，根据您提供的知识库，关于工商业储能的收益模式、成本、商业模式及充放策略，我为您梳理如下：

### 一、储能怎么赚钱？（主要收益模式）

储能的核心盈利逻辑&#x662F;**“低买高卖”**&#x548C;**“提供服务”**。

1. **峰谷套利（最核心、最稳定）**：
   * **原理**：利用电网分时电价机制，在电价低的**谷时段**充电，在电价高的**峰时段/尖峰时段**放电，赚取价差收益。
   * **地位**：这是用户侧储能最主要、最直接的收益来源。
2. **需量管理（降容降需）**：
   * **原理**：通过储能放电来“削峰”，降低企业用电的最高负荷（最大需量），从而减少每月按变压器容量或最大需量缴纳的基本电费。
   * **适用**：实行两部制电价的大工业用户。
3. **需求响应（参与电网互动）**：
   * **原理**：在电网用电紧张时，响应电网邀约，主动降低用电负荷（放电），获得电网支付的需求响应补贴。
   * **收益**：单次响应补贴可观，但具有偶发性。
4. **虚拟电厂（聚合增值）**：
   * **原理**：聚合储能、光伏、空调等可调负荷，通过虚拟电厂平台统一调度参与电力市场交易（调峰、调频等），获取额外收益。这是未来的重要增长点。
5. **应急备电（附加价值）**：
   * **原理**：作为备用电源，在市电中断时为重要负荷（如电梯、照明、消防）供电，减少断电损失。

### 三、成本多少？（投资成本参考）

储能系统的成本主要由电池、PCS（储能变流器）、BMS（电池管理系统）、EMS（能量管理系统）及配套工程组成。成本受技术路线、市场行情、项目规模影响。

* **磷酸铁锂电池**：占据市场主流（约90%份额），安全性高、成本低、循环寿命长。
* **2小时储能系统EPC成本**：约 **0.8-1.1元/Wh**。
* **2023年锂电储能系统均价**：约 **1.1元/Wh**。
* **以15MW/30MWh项目为例**：总投资约**2100万元**（折合0.7元/Wh）。

### 四、商业模式是什么样的？

| **模式**          | **投资方** | **收益方**    | **关键特点**         | **适用对象**          |
| --------------- | ------- | ---------- | ---------------- | ----------------- |
| **业主自投**        | 工商业用户   | 用户独享全部收益   | 收益最大、但需承担初始投资成本  | 资金充裕、有长远规划的企业     |
| **合同能源管理(EMC)** | 节能服务公司  | 用户享电价优惠/分成 | **零投资**、零风险，收益稳定 | 不愿出资、希望简化管理的企业    |
| **融资租赁+EMC**    | 融资租赁公司  | 用户分期付款/分成  | 降低初期资金压力         | 有固定资产、希望降低首付比例的企业 |

**主流模式（EMC）案例**：

* **华核绿能**：全额投资、运维，企业与用户分享节能收益。
* **上海勇绿新能源**：设立项目公司，用户提供场地，享受电价优惠。
* **新奥泛能**：投资建设储能设施，与用户签订EMC合同，从节能效益中回收投资。

### 五、什么时候充？什么时候放？（充放策略）

充放策略是储能盈利的核&#x5FC3;**，需根据当地的**分时电价政策**和**企业负荷曲线**来动态优化。通常采用“**&#x4E24;充两放**”策略以实现收益最大化。

**以广东省（非7-9月）为例：**

* **第一次循环：**
  * **充电**：**00:00 - 08:00**（谷段，利用低价电）。
  * **放电**：**10:00 - 12:00**（峰段，赚取峰谷差价）。
* **第二次循环：**
  * **充电**：**12:00 - 14:00**（平段，电价较低，补充电量）。
  * **放电**：**14:00 - 19:00**（峰段，再次赚取差价）。

**以山东省为例（深谷电政策）：**

* **冬季/春季/秋季**：可实现**深谷-尖峰**的极高价差套利，在深谷时段（如11:00-14:00）充电，在尖峰时段（如16:00-21:00）放电。
* **夏季**：结合空调负荷特性，优化充放时段，实现**平-尖**或**谷-尖**套利。

**核心原则**：

* **“低充高放”**：这是铁律。
* **动态调整**：能量管理系统（EMS）会根据**电价预测、负荷预测和天气数据**，实时优化充放电策略，确保收益最大化、同时保护电池寿命。
* **不超容、不反送**：充放电功率和时机需严格匹配变压器容量，避免产生额外的需量电费或违规反送电。

# 储能在取消行政分时后的盈利模式与现货市场分析

根据知识库资料，随着电力市场化改革深入，**直接参与市场交易的经营主体将不再人为规定分时电价水平和时段**（来源：《电力中长期市场基本规则》）。这意味着传统的“行政分时电价”逐步退出，储能盈利模式必须转向基于**电力现货市场信号**的动态套利。

## 一、取消行政分时后，储能的赚钱模式

### 1. 核心模式：现货市场套利（取代行政峰谷套利）

储能利用**现货市场价格的实时波动**进行”低价买入、高价卖出”的操作。在行政分时取消后，电价不再按固定时段划分，而是由**供需关系实时决定**，储能需要更精细的策略。

**参考收益数据（电力现货交易模式）：**

* **安捷物联**案例显示，现货市场电价预测准确率在**95%以上**，价差方向预测准确率约**60%**，可实现有效的日内套利。

![](attachment/1.png)

* **竞价策略**：即使电价方向预测错误（实际涨跌与预测相反），通过风险控制算法仍能实现盈利，有效降低亏损风险。

### 2. 虚拟电厂聚合交易

储能作为可调资源，被聚合至虚拟电厂平台，参与：

* **辅助服务市场**（调峰、调频）
* **需求响应**（电网应急调用）
* **电力现货交易**（日前、日内、实时市场）

**虚拟电厂收益来源：**

| **业务维度** | **收益模式** | **参考数值**                 |
| -------- | -------- | ------------------------ |
| 发电侧聚合    | 交易佣金     | 1%-3%                    |
| 电网侧调节    | 辅助服务补偿   | 按调节性能定价                  |
| 用户侧能效    | 需求响应补贴   | 削峰≤4500元/MWh，填谷≤120元/MWh |

### 3. 日内/实时市场频繁套利

**非固定时段、全天候响应**——储能不再局限于“谷充峰放”，而是：

* 根据**每15分钟**的现货价格信号，灵活决定充放时机
* 一天内可进行多次充放（不再受“两充两放”的传统策略限制）
* 可利用日内价格的**微小波动（波动区间≥价差阈值即可）**&#x8FDB;行高频低收益套利

## 二、什么情况下现货价格高/低？

### 📈 现货价格高的典型情况

| **场景**      | **原因**            | **典型时段**                   |
| ----------- | ----------------- | -------------------------- |
| **用电高峰时段**  | 负荷集中增加，供不应求       | 工作日10:00-12:00、14:00-19:00 |
| **新能源出力不足** | 阴天/无风，光伏/风电出力大幅下降 | 冬季傍晚（17:00-20:00）          |
| **极端天气**    | 空调负荷激增，电网承压       | 夏季35℃以上高温天                 |
| **非预期停机**   | 火电/核电等大型机组故障跳闸    | 随机事件，需实时监测                 |
| **日前预测偏差**  | 实际负荷高于预测，实时市场补仓   | 实时市场结算前                    |

### 📉 现货价格低的典型情况

| **场景**       | **原因**              | **典型时段**                    |
| ------------ | ------------------- | --------------------------- |
| **新能源大发**    | 光伏“午间高峰”大量集中并网，供给过剩 | 晴天的11:00-14:00（山东曾出现0价甚至负价） |
| **用电低谷期**    | 凌晨工业用电少、居民用电少       | 00:00-06:00                 |
| **节假日/周末**   | 工商业停休，负荷骤降          | 春节、国庆等                      |
| **风电满发+低负荷** | 夜间风力发电强+社会用电少       | 深夜至凌晨                       |

> **山东特别情况（2025年上半年）：** 光伏发电结算均价仅约**0.07元/kWh**，说明新能源大发时段现货价格极低，储能可获得极大的“谷-峰”价差。

## 三、现货价格的变化时间：每15分钟一个价格

现货市场的核心特征是高时间分辨率：

### 📊 交易时间粒度

| **市场类型** | **出清周期**               | **说明**                     |
| -------- | ---------------------- | -------------------------- |
| **日前市场** | 每**15分钟**一个价格点（96个点/天） | 在运行日前一天，预测并确定次日96个时段的价格    |
| **日内市场** | 每**15分钟**滚动出清          | 修正日前预测偏差，应对负荷突变（大部分省份暂未开展） |
| **实时市场** | 运行前**5分钟至1小时内**完成出清    | 最终结算价，精准平衡实时供需偏差           |

### ⏱ 交易时间轴图示

```
日前市场 → 日内市场 → 实时市场
（D-1日预测） （D日滚动修正） （D日最终出清）

每15分钟一个价格节点
00:00 → 00:15 → 00:30 → 00:45 → ... → 23:45
  ↓        ↓        ↓        ↓            ↓
P₁       P₂       P₃       P₄           P₉₆
```

**结论**：现货价格**每15分钟变化一次**，一天共有96个价格节点。

## 四、现货价格的影响因素

### 1. 供需基本面（核心因素）

* **发电侧**：新能源出力占比（光伏/风电）、火电/核电出力、水电来水情况
* **用电侧**：气温（空调负荷）、节假日、经济景气度、生产计划

### 2. 新能源随机性与预测误差

* 光伏、风电具有**强随机性、间歇性、波动性**
* 预测偏差会导致日前-实时价差拉大，**偏差越大，储能套利空间越大**

### 3. 电网约束与阻塞

* 局部输电通道拥堵，导致该区域电力“送不出、进不来”
* 区域价格出现**严重的时空差异性**（同一时间不同地点价格不同）

### 4. 燃料成本与碳价

* 天然气/煤炭价格上涨 → 火电边际成本增加 → 抬高整体电价水平
* 碳排放配额价格波动也影响火电报价

### 5. 政策与市场规则

* 辅助服务市场补偿机制
* 需求响应补贴标准
* 市场力监测与限价机制

### 6. 市场预期与投机行为

* 发电企业报价策略
* 天气预测（如台风、寒潮）的预期化交易
* 节前备货/节后复工的预期性波动

## 🔑 核心结论

| **维度**            | **具体内容**                          |
| ----------------- | --------------------------------- |
| **取消行政分时后储能赚钱核心** | 依赖**现货市场的实时价差套利 + 辅助服务/虚拟电厂聚合增值** |
| **高价格场景**         | 用电高峰、新能源出力不足、极端天气、非预期停机           |
| **低价格场景**         | 新能源大发（午间光伏）、用电低谷（凌晨）、节假日、负电价      |
| **价格变化频率**        | **每15分钟一个价格**（一天96个价格节点）          |
| **核心影响因素**        | 供需平衡度、新能源预测偏差、电网阻塞、燃料碳价、政策规则      |

**储能运营核心能力（参考安捷物联数据）：**

* **电价预测准确率**≥95%（预测“数值”）
* **涨跌方向预测准确率**≈60%（预测“涨还是跌”）
* **通过风险控制算法**，即使在方向预测错误时也能实现盈利
